# typora 快捷键

1. 插入表格：`command + T`

2. 行内代码：`` `

3. 行间代码：`command + option + C`  或者 ` ``` `

4. 行内公式：`$[公式]$`

5. 行间公式：`$$[公式]$$` 或者 `$$ + enter`  或者 `command + option + B`

6. 选中单词：`command + d`

7. 选中一行：`command + l`

8. 搜索：`command + f`

9. 替换：`command + h`

10. 插入链接：`command + k`

11. 流程图

    ```flow
    st=>start: 开始
    op=>operation: 我的操作
    cond=>condition: 确认？
    io=>inputoutput: 输入输出
    sub=>subroutine: 子任务
    e=>end: 结束
    st->op->io->sub->cond
    cond(yes)->e
    cond(no)->op
    ```
